üìñ Introduction

YURI BOT adalah Discord bot all-in-one yang dirancang untuk membuat server Discord Anda lebih hidup, terorganisir, dan menyenangkan. Dengan ratusan fitur yang powerful namun mudah digunakan, YURI BOT adalah solusi sempurna untuk server Discord dari berbagai ukuran.

Bot ini dilengkapi dengan sistem music premium 24/7, leveling system dengan rank card yang keren, economy system yang kompleks, moderation tools yang canggih, dan masih banyak lagi. Semua fitur dirancang dengan fokus pada kemudahan penggunaan dan performa yang optimal.

üí°
Pro Tip

Mulai dengan menjalankan command /help untuk melihat semua fitur yang tersedia dan /setup untuk konfigurasi awal yang cepat!

‚öôÔ∏è Setup Guide

Ikuti langkah-langkah berikut untuk mengundang dan mengatur YURI BOT di server Discord Anda:

  1. Undang Bot ke Server

    Klik tombol "Invite Bot" di halaman utama atau gunakan link invite resmi. Pastikan Anda memiliki permission "Manage Server" untuk mengundang bot. Pilih server yang ingin Anda tambahkan dan klik "Authorize".

  2. Atur Permissions

    YURI BOT memerlukan beberapa permission untuk berfungsi optimal. Permission yang diperlukan: Send Messages, Embed Links, Manage Messages, Connect, Speak, dan Manage Roles (untuk auto-role feature).

  3. Jalankan Setup Wizard

    Gunakan command /setup untuk menjalankan setup wizard interaktif. Wizard ini akan membantu Anda mengkonfigurasi fitur-fitur utama seperti music channels, leveling system, welcome messages, dan lainnya dengan mudah.

  4. Kustomisasi Settings

    Setelah setup dasar selesai, gunakan command /config untuk mengakses panel konfigurasi lengkap. Di sini Anda bisa mengatur prefix custom, mengaktifkan/nonaktifkan fitur tertentu, dan menyesuaikan bot sesuai kebutuhan server Anda.

  5. Selesai!

    Bot sudah siap digunakan! Coba command /help untuk melihat semua fitur yang tersedia, atau langsung mulai dengan /play [song name] untuk memutar musik.

‚ú® Features Overview

üéµ Music System

Sistem musik premium dengan support untuk YouTube, Spotify, dan SoundCloud. Fitur termasuk queue management, loop mode, shuffle, volume control, bass boost, dan equalizer settings. Music player dapat berjalan 24/7 tanpa disconnect.

‚≠ê Leveling System

Sistem leveling otomatis yang memberikan XP setiap kali user chat atau join voice channel. Dilengkapi dengan animated rank card yang bisa dikustomisasi, global leaderboard, dan reward berupa auto-role saat mencapai level tertentu.

üí∞ Economy System

Virtual economy lengkap dengan currency, bank system, daily/weekly rewards, work commands, shop untuk membeli items, gambling games (slots, coinflip, blackjack), dan transfer system antar users.

üõ°Ô∏è Moderation Tools

Tools moderasi lengkap termasuk auto-moderation (anti-spam, anti-link, bad words filter), ban/kick/mute commands, warning system, raid protection, dan detailed logging untuk semua moderation actions.

‚ùì Frequently Asked Questions

Apakah YURI BOT gratis?

Ya, YURI BOT 100% gratis untuk semua fitur dasar! Kami juga menawarkan Premium tier dengan fitur tambahan seperti custom bot branding, priority support, dan increased limits.

Bagaimana cara mendapatkan support?

Join Discord support server kami atau buat ticket menggunakan command /ticket. Tim support kami siap membantu 24/7!

Apakah data saya aman?

Sangat aman! Kami menggunakan enkripsi untuk semua data sensitif dan tidak pernah menjual data user kepada pihak ketiga. Baca Privacy Policy kami untuk detail lengkap.

Bot tidak merespon, kenapa?

Pastikan bot memiliki permission yang cukup dan role bot berada di posisi yang tepat. Jika masih bermasalah, coba kick dan invite ulang bot, atau hubungi support kami.
