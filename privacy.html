PRIVACY POLICY

Kami menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan YURI BOT.

01

Informasi yang Kami Kumpulkan

YURI BOT mengumpulkan berbagai jenis informasi untuk memberikan layanan terbaik kepada pengguna kami. Berikut adalah informasi yang kami kumpulkan:

  • User ID Discord: ID unik Anda dari Discord untuk identifikasi pengguna
  • Server ID: ID server Discord tempat bot digunakan
  • Pesan & Interaksi: Command yang Anda gunakan dan interaksi dengan bot
  • Data Leveling: XP, level, dan statistik aktivitas Anda
  • Data Economy: Saldo virtual, transaksi, dan item yang dimiliki
  • Preferensi: Pengaturan custom dan konfigurasi bot yang Anda buat
  • Log Aktivitas: Riwayat penggunaan command untuk analisis dan perbaikan

02

Bagaimana Kami Menggunakan Data

Data yang kami kumpulkan digunakan secara bertanggung jawab untuk tujuan berikut:

  • Penyediaan Layanan: Menjalankan fitur-fitur bot seperti leveling, music, economy, dan moderasi
  • Personalisasi: Menyimpan preferensi dan pengaturan custom Anda
  • Peningkatan Layanan: Analisis penggunaan untuk mengembangkan fitur baru
  • Keamanan: Mendeteksi dan mencegah penyalahgunaan atau pelanggaran
  • Support: Membantu menyelesaikan masalah teknis yang Anda laporkan
  • Komunikasi: Memberitahu Anda tentang update penting atau perubahan layanan

Kami HANYA menggunakan data untuk keperluan operasional bot dan peningkatan layanan. Data Anda TIDAK AKAN PERNAH dijual ke pihak ketiga untuk tujuan marketing atau komersial.

03

Penyimpanan & Keamanan Data

Kami sangat serius dalam melindungi data Anda dengan berbagai langkah keamanan:

  • Enkripsi: Data sensitif dienkripsi saat disimpan dan ditransmisikan
  • Server Aman: Database kami disimpan di server dengan proteksi berlapis
  • Akses Terbatas: Hanya tim developer yang berwenang yang dapat mengakses database
  • Backup Rutin: Data di-backup secara teratur untuk mencegah kehilangan
  • Monitoring 24/7: Sistem keamanan dipantau terus-menerus
  • Update Berkala: Patch keamanan diterapkan secara proaktif

Berbagi Data dengan Pihak Ketiga

  • Analytics Tools: Statistik anonim untuk memahami penggunaan (tanpa data pribadi)
  • Ketentuan Hukum: Jika diwajibkan oleh hukum atau proses hukum yang sah

Hak-Hak Pengguna

Sebagai pengguna YURI BOT, Anda memiliki hak-hak berikut terkait data pribadi Anda:

  • Hak Akses: Meminta salinan data pribadi yang kami miliki tentang Anda
  • Hak Koreksi: Meminta perbaikan data yang tidak akurat atau tidak lengkap
  • Hak Penghapusan: Meminta penghapusan data pribadi Anda dari sistem kami
  • Hak Pembatasan: Membatasi cara kami memproses data Anda
  • Hak Portabilitas: Menerima data Anda dalam format yang mudah dibaca
  • Hak Penarikan: Menarik persetujuan penggunaan data kapan saja

Cara Menggunakan Hak Anda

06

Cookies & Teknologi Tracking

YURI BOT adalah aplikasi Discord bot dan tidak menggunakan cookies browser tradisional. Namun, kami menggunakan teknologi berikut:

  • Session Storage: Untuk menjaga koneksi aktif selama Anda menggunakan bot
  • Cache Data: Menyimpan sementara data untuk meningkatkan performa
  • Analytics Tokens: Identifikasi anonim untuk statistik penggunaan

Semua teknologi ini digunakan hanya untuk keperluan fungsional bot dan tidak untuk tracking perilaku personal Anda di luar interaksi dengan YURI BOT.

YURI BOT dirancang sebagai platform yang aman untuk semua usia. Namun, kami memiliki kebijakan khusus terkait privasi anak-anak:

  • Kami tidak secara sengaja mengumpulkan data dari anak di bawah 13 tahun tanpa izin orang tua
  • Jika kami mengetahui telah mengumpulkan data dari anak di bawah 13 tahun, kami akan segera menghapusnya
  • Orang tua dapat menghubungi kami untuk meninjau, menghapus, atau melarang pengumpulan data anak mereka
