Frequently Asked Questions

Bagaimana cara invite YURI BOT ke server saya? +
Klik tombol "Invite Bot" di halaman utama atau gunakan link invite yang tersedia. Pastikan kamu memiliki permission "Manage Server" untuk invite bot. Setelah invite, ikuti setup wizard untuk konfigurasi awal.
Apa prefix default dari YURI BOT? +
Prefix default adalah "!" (tanda seru). Kamu bisa mengubah prefix menggunakan command !setprefix [new prefix]. Selain itu, kamu juga bisa mention bot sebagai alternatif prefix.
Bagaimana cara mendapatkan premium? +
Kunjungi halaman Premium untuk melihat paket yang tersedia. Kamu bisa subscribe melalui website kami atau langsung di Discord server menggunakan command !premium. Premium memberikan akses ke fitur eksklusif dan priority support.
Bot tidak merespon command saya, kenapa? +
Pastikan: 1) Bot memiliki permission yang cukup di channel tersebut, 2) Kamu menggunakan prefix yang benar, 3) Command yang kamu gunakan tersedia (cek !help), 4) Bot tidak sedang maintenance. Jika masih bermasalah, hubungi support kami.
Apakah YURI BOT gratis? +
Ya! YURI BOT sepenuhnya gratis untuk digunakan dengan akses ke semua fitur dasar. Kami juga menawarkan paket Premium untuk fitur tambahan dan priority support, namun ini bersifat opsional.
Bagaimana cara melaporkan bug atau memberikan saran? +
Join Discord support server dan gunakan channel #bug-report untuk melaporkan bug, atau #suggestions untuk memberikan saran. Kamu juga bisa mengirim email ke support@testbot.com dengan detail lengkap.
